**Тема и концепция**

Тема приложения – медицинский центр. 

Приложение медицинского центра позволит пациентам быстро записываться на приём к врачу в любое время суток, не звоня в регистратуру. Также у клиента будет быстрый доступ ко всем его записям, что помогает не забывать о приёме. Благодарю данному приложению администрация медцентра может легко просматривать загруженность врачей и их график.
Целевой аудиторией приложения являются люди всех возрастов, которым требуется помощь в области здоровья и медицины.
Ключевой функционал: клиент может просмотреть список всех предоставляемых услуг с их стоимостью, а также список врачей, после чего он может выбрать дату и записаться на приём.

Структура приложения

1.	Главная (/)

2.	Аутентификация (/auth)

3.	Личный кабинет пользователя (/profile)

4.	Админ-панель (/admin)

5.	Вкладка «О нас» (/aboutus)

6.	Каталог услуг с поиском (/uslugi)

7.	Каталог врачей с поиском (/doctors)

8.	Список акций (/aktsii) 

9.	Список отзывов клиентов (/otzivi)
